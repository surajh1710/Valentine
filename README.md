# Valentine Interactive Page

A cute, playful, interactive Valentine “ask” page with mini-steps and a fun finale.

## Quick Start

Open `index.html` in your browser. That’s it.

## Customize

- Update the name in `index.html`:
  - Find `id="herName"` and replace **Love** with her name.
- Tweak date options in `index.html` (search for `data-choice=`).
- Change colors in `styles.css` (look for the `:root` section).

## Optional: Local Preview Server

If your browser blocks some features from local files, run a simple server:

```powershell
# From this folder
python -m http.server 8000
```

Then visit `http://localhost:8000`.
